# C Calculator 
A simple calculator application built using C programming language.
## FEATURES
Addition
Subtraction 
Multiplication 
Division 
user friendly console interface 

## TECHNOLOGIES USED
C programming Language 
GCC Compiler 
Visual Studio Code

## HOW TO RUN
1. compile program:
 gcc calculator.c -o calculator
2. Run the program:
 ./calculator
